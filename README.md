# Geng Shii

the project is a terminal-based adventure game built for the web. it operates as a state-driven narrative experience, focusing on systemic rebellion within a dystopian institution.

## overview

Geng Shii is developed by Quing. the application utilizes react and typescript to create a high-fidelity text interface. the architecture maintains a strict separation between game logic and the presentation layer.

## technical stack

the system is built on the following fundamentals:

- **runtime:** bun
- **framework:** react 19
- **language:** typescript 6
- **bundler:** vite 8
- **styling:** vanilla css

## installation

to initialize the development environment, execute the following commands:

```bash
bun install
bun run dev
```
